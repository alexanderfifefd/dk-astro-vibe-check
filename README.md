# Datakollektivet

Website for Datakollektivet — a Norwegian non-profit for open data and digital literacy.

Built with [Astro](https://astro.build/) v5, [Tailwind CSS](https://tailwindcss.com/) v4, and [React](https://react.dev/) (for interactive components).

## Getting started

```bash
npm install
npm run dev
```

## Project structure

```
src/
├── components/     # Astro + React components
├── content/        # Markdown content collections (services)
├── data/           # JSON data (chronicle timeline)
├── layouts/        # Page layout
├── pages/          # Routes
└── styles/         # Global CSS + Tailwind theme
docs/               # Project documentation
```

See `docs/overview.md` for the full project overview.
